# Contributing

This database grows with the community. There are three ways in:

## Report abusive IPs

- **Fastest:** [ffraud.com/report](https://ffraud.com/report). One IP or a bulk list, with the category (brute-force, spam, phishing, c2, scanner, web-attack, botnet, bot, scraping, fraud) and any evidence you have (log excerpt, gist, screenshot).
- **Or open an issue** here with the IPs and what they did.

Every report is verified before it ships: an IP enters the database once it is confirmed by at least two independent monitoring networks or by verified community evidence. Confirmed reports land in the next build, with credit on the [leaderboard](https://ffraud.com/reporters) if you want it.

## Dispute a listing

Run a network or an IP listed here and believe the listing is wrong? [Tell us](https://ffraud.com/report) or open an issue. We recheck on the next build and drop entries that no longer hold.

## Improve the data

Ideas for new fields, formats (JSON, Parquet, MMDB), or delivery channels? Open an issue. The data is MIT, so you can always fork and remix, but if something would help everyone, we would rather ship it here for everybody.
